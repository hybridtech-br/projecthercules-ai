# Project Hercules AI

Privacy-First AI Assistant for Home Assistant.

Project Hercules AI is an open-source assistant designed exclusively for Home Assistant.

## Features

- Wake phrase support ("Hi Hercules", "Oi Hercules")
- Multi-language support based on device language
- Secure architectural isolation
- Add-on based installation
- FastAPI backend

## Privacy by Architecture

Project Hercules AI:

- Does NOT access other mobile apps
- Does NOT access device files
- Does NOT execute OS-level commands
- Operates strictly via Home Assistant API

## Installation

1. Add this repository to Home Assistant Add-on Store:
   https://github.com/SEU_USUARIO/projecthercules-ai

2. Install the "Project Hercules AI" add-on.

3. Start the add-on.

4. Access:
   http://YOUR_HA_IP:8099
